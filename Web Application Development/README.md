## Course Description
This course serves as an applied introduction to designing and developing web applications. 

Topics to be covered in this course include: 
* Web architectures
* Client-side design and interactivity
* Server-side web page generation
* Accessing and updating database data, using web services/APIs, XML
* Web security


## Learning Outcomes
On the successful completion of the course, students will be able to: 
* Create web applications using HTML5 
* Design a web-based user interface, and apply styling and layout to that user interface 
* Create interactive client-side elements for a web application with JavaScript 
* Use popular JavaScript-based libraries, such as D3.js or jQuery 
* Implement server-side behaviour using a modern library, such as Node.js 
* Save and access data from a persistent store, such as PosgreSQL or MongoDB
